class src.var.grid#

Grid class

__init__()#

Contains the (lat,lon) of each triangular grid cell with the corresponding vertices (lat_1, lat_2, lat_3), (lon_1, lon_2, lon_3).

link is a lookup table linking the grid cell to the corresponding topography file.
